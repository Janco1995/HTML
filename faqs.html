Frequently Asked Questions

Is HTML the only markup language?

No, HTML is not the only markup language. There are other markup languages such as XML (Extensible Markup Language), CSS (Cascading Style Sheets) and JavaScript. Modern web pages use a combination of these markup language to be as effective as possible.

What is the difference between HTML, CSS and JavaScript

HTML is used to create the basic structure and content of a webpage.
CSS is used for designing the webpage - where everything is placed and how it looks.
JavaScript is used to define the interactive elements of the webpage that promote engagement.

How do I view HTML files?

Double-click on the HTML file. The file will open in your default browser. The file will immediately be interpreted and displayed as instructed (without the markup tags).

How do I edit HTML files?

To open an HTML file to make changes to the code, right-click on the file and select edit with Notepad/Notepad++. If this option is not available, scroll to open with and select default application as Notepad/Notepad++
